## Running the tests

Install Karma:

```bash
$ npm install -g karma
```

Now we can run the tests with:

```bash
$ karma start
```

If you didn't configure to watch all the files and run tests
automatically on any change, you can trigger the tests manually by
typing:

```bash
$ karma run
```
